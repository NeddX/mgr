◆ ConnectToRC()

Result< Err > pmgrd::Application::ConnectToRC ( )
noexcept

Tries to connect to the RC server.

After successfully connecting, this method sends a net::PacketType::InitCon packet which contains its ID from /etc/vlink.conf to the RC to register itself as an Endpoint on the RC side. Afterwards it sends a net::PacketType::GetConfig where the RC tries to match it with a Crew Station and then responds with a json object containing its Camera.

Returns
Result of Err where Err indicates an error has occured.
